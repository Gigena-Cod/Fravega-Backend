# PROYECTO FRÁVEGA

API https://fravega-backend.herokuapp.com/

## PRODUCTO (endpoint)

- Obtener todos los productos
GET /getProducts

- Obtener producto por id
GET /getProduct/:id

## USUARIO (endpoint)

- Obtener todos los usuarios
GET /getUsers

- Obtener usuario por id
GET /getUser/:id

## PERFIL (endpoint)

- Obtener todos los perfiles
GET /getProfiles

- Obtener perfil por id
GET /getProfile/:id

## ORDEN (endpoint)

- Obtener todos los perfiles
GET /getOrders

- Obtener perfil por id
GET /getOrder/:id

## DIRECCIÓN (endpoint)

- Obtener todos los perfiles
GET /getDirections

- Obtener perfil por id
GET /getDirection/:id
